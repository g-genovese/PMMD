# `data/` — result datasets and parameters

This folder holds (a) **result digests** of the framework's headline numbers — each
with its provenance (the generating script and the paper remark/theorem it backs) —
and (b) small raw parameter/output files from specific runs. The **full** distributed
raw outputs (percolation sweeps, the `d_BS` ensembles, the 3-fold-merger events, the
Koide `L`-scan) are large and are **regenerated by the scripts** in `src/` (and the
launchers in `scripts/`); see [`../docs/SCRIPTS.md`](../docs/SCRIPTS.md).

## Result digests (headline numbers, with provenance)
| File | Content | Backs |
|------|---------|-------|
| `results_dBS.json`       | causal dimension `d_BS` by method (mature foam → 4; retired `p_c` probe → 3.6) | Remarks `dBS-cutproject-v53`, `dBS-growth-v53` |
| `results_koide.json`     | Koide `Q_K` operators (continuum `conn → 0.674`), soliton zero-modes, angle `θ=2/9` | Remark `gauge-overlap-generations-v53`, Thm `koide-identity` |
| `results_cosmology.json` | `Λ` (three identifications), `η_B`, `Ω_DM/Ω_B`, PBH spectrum, `n_γ/n_B` | §`Lambda-refinement`, Thm `omega-DM-heavy-tailed` |
| `results_chain.json`     | the chain, the minimal unit (4 nodes = 4 states = 1 qubit), the two routes to `E_8`, key invariants | Prop `v6-C3-phase`, Thm `mckay` |

## Raw run files
- `option_A_FSS_results.json` — the `1/p_c` FSS extrapolation (per-`L` means/SEMs, fits, `f_c·p_c` discrimination) from `src/phenomenology/option_A_FSS_extrapolation.py`.
- `L12_pc_refined.json` — the `L=12` percolation run (`N≈4.3×10⁸`, 32 trials, Newman–Ziff) from `src/e8_structure/e8_percolation_implicit_v42.py`.
- `*.npz` — small NumPy artefacts from the flavour / PMNS / case-(G) probes in `src/flavour/` (basins, frontiers, phase closures, holonomies).

> The figures in [`../figures/`](../figures/) are generated from these numbers; see `../docs/SCRIPTS.md` for the figure list.
